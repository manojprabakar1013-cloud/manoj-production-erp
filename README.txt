Manoj Production ERP Website
============================

How to use:
1. Open index.html in Chrome / Edge.
2. Enter Job Master first.
3. Enter Tag Master if job has tags.
4. Add Daily Entry with worker name, start time and end time.
5. Worker hours and OT are calculated automatically.
6. Use CEO Report page to generate daily update text.
7. Use Backup JSON regularly to save data.

Default Time:
Regular time: 08:30 AM to 05:00 PM
OT: After 05:00 PM

Hosting:
You can upload the index.html file to GitHub Pages, Netlify, Vercel, or any normal web hosting.

Important:
This is a static website. Data is stored in the browser local storage. For multi-user/company usage, a backend database should be added later.
